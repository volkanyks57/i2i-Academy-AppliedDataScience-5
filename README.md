# i2i-Academy-AppliedDataScience-5
Heart Disease veri seti üzerinde 4 farklı classification algoritmasının karşılaştırmalı analizi

# Özet
  Bu projede Heart Disease veri seti kullanılarak 4 farklı classification algoritması
  karşılaştırmalı olarak analiz edildi. Veri temizleme, Feature Scaling ve model
  değerlendirme adımlarını kapsayan uçtan uca bir Machine Learning pipeline oluşturuldu.

# Tamamlanan Görevler
  - Heart Disease veri seti yüklendi ve incelendi
  - Eksik değerler ve tekrar eden satırlar temizlendi
  - Veri %80 eğitim, %20 test olarak bölündü ve Feature Scaling uygulandı
  - K-Nearest Neighbors, Random Forest, SVM ve Logistic Regression modelleri eğitildi
  - Her model için Accuracy Score ve Confusion Matrix hesaplandı
  - Modeller karşılaştırılarak en iyi performans gösteren model belirlendi

# Sonuçlar
  - K-Nearest Neighbors = %73.77
  - Random Forest = %83.61
  - SVM = %78.69
  - Logistic Regression = %77.05
